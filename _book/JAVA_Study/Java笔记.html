N.SomeQuestion

3.1 出现乱码

原因： 编码格式不一致
解决方案： conf/logging.properties
All UTF-8 替换为GBK
java.util.logging.ConsoleHandler.encoding = UTF-8

MAC下 server.xml 中

<Connector port="8080" protocol="HTTP/1.1"
               connectionTimeout="20000"
               redirectPort="8443" URIEncoding="UTF-8"/>

成功

3.2 出现一闪而过的黑屏

原 因： Java环境变量设置错误
解决方案： 重新设置环境变量
